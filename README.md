# SnakeWaterGun_Game
It's a game.....
# Description
In this game there are three option in this game .User can select either one of them from water,snake,gun and play with the computer.There are ten rounds in this game.
# Rules
YOU VS COMPUTER
GUN VS GUN = DRAW,
GUN VS SNAKE = WIN,
GUN VS WATER = LOSE,
WATER VS WATER = DRAW,
WATER VS GUN = WIN,
WATER VS SNAKE = LOSE,
SNAKE VS SNAKE = DRAW,
SNAKE VS WATER = WIN,
SNAKE VS GUN = LOSE
